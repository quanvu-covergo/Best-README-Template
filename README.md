
## Installation
### Via npm
```bash
    npm i
```
### Via yarn
```bash
    yarn
```

## Run
### Via npm
```bash
    npm run dev
```
### Via yarn
```bash
    yarn dev
```

## Build
### Via npm
```bash
    npm run build
```
### Via yarn
```bash
    yarn build
```

## Preview Production
### Via npm
```bash
    npm run serve
```
### Via yarn
```bash
    yarn serve
```

## Description

We should use \*ant-icon\* instead of \*a-icon\*

```
<a-icon type="user" /> => <ant-icon type="user" />
```
