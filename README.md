
# CoverAPP

## Installation
### Via npm
```bash
    npm i
```
### Via yarn
```bash
    yarn
```

## Run
### Via npm
```bash
    npm run dev
```
### Via yarn
```bash
    yarn dev
```

## Build
### Via npm
```bash
    npm run build
```
### Via yarn
```bash
    yarn build
```

## Preview Production
### Via npm
```bash
    npm run serve
```
### Via yarn
```bash
    yarn serve
```

## Descriptions

We should use \*ant-icon\* instead of \*a-icon\*

- False
    ```JS
    <a-icon type="user" />
    ```

- True
    ```JS
    <ant-icon type="user" />
    ```
